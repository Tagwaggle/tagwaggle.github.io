>_ Taniti Website Project

Project Overview

The Taniti website was designed as a fictional town site, offering visitors information on places to visit, stay, and how to navigate the town. I created the wireframes and built the website using Webflow, ensuring it was visually engaging yet easy to navigate.

Summary of Usability Feedback

The overall feedback from the users was positive, with some issues regarding navigation, layout, and naming conventions. Below are some key insights:

  • Reviewer 1: Positive experience overall, but found navigation and naming conventions confusing. Suggestions for cleaning up navigation and clarifying text.
  • Reviewer 2: Reported issues with dropdown menu and pilot information clarity. Suggested adjusting the layout to improve flow.
  • Reviewer 3: Enjoyed the site overall, but had issues with landing page directing to the wrong page and confusion over contact information.

Design Change Process

The initial top-down design approach was found to be ineffective after user testing. We decided to restructure the flow so that each subpage could be accessed directly from the homepage. Additionally, based on feedback, we added a contact page and decided to separate the FAQ and Contact Us pages for better clarity.
